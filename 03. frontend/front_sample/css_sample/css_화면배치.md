## 1. float

## 2. position

    1. static
    - 기본 포지션
    - 배치하는 순서대로 위치

    2. relative
    - 원래 위치를 기준으로 width, height 만큼 위치 이동(현재 요소 기준)
    - 상대 경로

    3. absolute
    - 내가 속한 상자를 기준으로 위치 이동
    - 절대 경로
    - 부모의 position:relative 이어야 한다.

    4. fixed
    - 윈도우 페이지 기준(view port) 위치
