## Vue.js 주요 Directive

- v-text
- v-html

- v-once : 변경이 필요하지 않는 딱 한번 나타낼 때
- v-show : 렌더링 되고 true/false에 따라서 show 여부가 결정된다 (display:none)
- v-if / v-else-if / v-else : false일시 렌더링도 되지 않는다 (visibility: hidden)

- v-for


- v-bind : 생략 가능. 속성명에 할건지 속성값에 할건지
- v-model 

- v-on : xxxEvent / v-on: click
